# POS UMKM - Aplikasi Kasir dengan Scan Barcode

Aplikasi Point of Sale untuk UMKM dengan fitur scan barcode produk distributor dan laporan keuangan cash flow otomatis.

## Teknologi
- Laravel 13
- MySQL 8.0
- Livewire 3
- Tailwind CSS
- HTML5 Camera API
